Comparing Technologies for User-Issued Assets

This website compares various Bitcoin 2.0 technologies.

Technologies not using the Bitcoin Blockchain (such as Ethereum, NXT or BitShares) are not presented here.

Some of these protocols have features not related to user-issued assets. For example, Counterparty has core protocol support for P2P betting and rock-paper-scissors, however this website only compares features relating to user-issued assets and their management.
